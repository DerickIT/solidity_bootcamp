# Carreer Bridge

## 1. 项目概述

- **项目名称** ：Carreer Bridge
- **项目简介**：建立求职者与用人方的信任桥梁
- **项目愿景**：旨在为用人方更好得到想要的人才，旨在为求职者更好的找到好工作

## 2. 技术架构

- **区块链类型**：以太坊+Polygon
- **技术框架**：ERC20，ERC721 Token

## 3. 核心功能（Website）

- **功能一**：学习进阶技术能力
- **功能二**：获得对应技术认证
- **功能三**: 生成简历并投递
